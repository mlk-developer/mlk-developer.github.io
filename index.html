Pourquoi le WEB?

Pour ma part, la filière qui m'intéresse le plus est la filière ingénierie du WEB car je trouve que le programme est complet, nous verrons le WEB dans son intégralité avec beaucoup de framework comme Laravel, Symfony, React JS ...
De plus, le WEB est un domaine très important aujourd'hui, pour l'utilité qu'elle apporte aux entreprises/particuliers.
Par ailleurs, c'est un secteur porteur qui est omniprésent dans l'international.
Enfin, créer, imaginer, concevoir un site fonctionnel, sont des atouts qui me correspondent et qui me permettront d'être opérationnel dans toutes les missions qui me seront confiées. D'ailleurs mes dires relèvent une question : "Pourquoi me prendre?"

Pourquoi moi?
